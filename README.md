# KCU 빅데이터 실습 - 연습 노트북

통계, 머신러닝, 데이터 분석을 다루는 빅데이터 실습 과정의 Jupyter 노트북 모음입니다.

## 주차별 실습 내용

### 3주차 - 통계와 확률
**파일:** `3주차실습.ipynb`
- 기초 통계량 (평균, 중앙값, 최빈값, 분산, 표준편차)
- 확률 시뮬레이션 및 분포 (정규분포, 이항분포, 푸아송분포)
- 점추정과 구간추정
- t-검정을 이용한 가설검정

### 4주차 - 데이터 전처리
**파일:** `4주차실습.ipynb`
- 결측치 처리 (삭제 및 대체)
- IQR 방법을 이용한 이상치 탐지 및 처리
- groupby를 활용한 데이터 집계

### 5주차 - 탐색적 데이터 분석 (EDA)
**파일:** `5주차실습.ipynb`
- 데이터셋: Titanic
- 데이터 구조 파악 및 이해
- 결측치 및 이상치 처리
- 수치형/범주형 변수 분석
- 변수 간 상관관계 분석

### 6주차 - 데이터 시각화
**파일:** `6주차실습.ipynb`
- 데이터셋: Titanic, Flights
- matplotlib vs seaborn 비교
- 단변수 그래프: 히스토그램, 박스플롯, KDE, 라인플롯, 막대그래프
- 다변수 그래프: 산점도, 회귀선, 페어플롯, 히트맵

### 7주차 - 분류 모델
**파일:** `7주차실습.ipynb`
- 데이터셋: 유방암(Breast Cancer)
- 분류 알고리즘: 로지스틱 회귀, 의사결정나무, 랜덤포레스트, KNN, 나이브베이즈, SVM
- 모델 성능 비교

### 9주차 - 회귀 모델
**파일:** `9주차실습.ipynb`
- 데이터셋: 보험료(Insurance)
- 단순 및 다중 선형 회귀
- 회귀 모델 비교: Linear, Ridge, Lasso, Decision Tree, Random Forest
- MAE, MSE, RMSE를 이용한 모델 평가

### 10주차 - 차원 축소 및 군집화
**파일:** `10주차실습.ipynb`
- 데이터셋: 와인(Wine)
- 차원 축소: PCA, t-SNE
- 군집화 알고리즘: K-means, 계층적 군집화, DBSCAN
- 엘보우 방법 및 실루엣 점수를 이용한 최적 군집 수 선택

### 11주차 - 자연어 처리 (한국어)
**파일:** `11주차실습.ipynb`
- 데이터셋: 네이버 영화 리뷰 감성 분석(NSMC)
- 한국어 텍스트 전처리 (정규화, 형태소 분석, 불용어 제거)
- 텍스트 벡터화: BoW, DTM, TF-IDF
- 나이브 베이즈를 이용한 감성 분류

### 12주차 - 모델 평가 및 LLM 활용 분석
**파일:** `12주차실습.ipynb`
- 데이터셋: 유방암, 심혈관 질환
- 평가 지표: 혼동행렬, 정밀도, 재현율, F1-Score, ROC-AUC
- LLM을 활용한 데이터 탐색 및 모델링
- 불균형 데이터 처리

### 13주차 - 건강검진 데이터 분석
**파일:** `13주차실습.ipynb`
- 데이터셋: 건강검진 2024 (100만 건)
- 대용량 데이터 처리 및 메모리 최적화
- 건강 지표 파생변수 생성
- 다양한 알고리즘을 활용한 당뇨 위험도 예측
- 언더샘플링을 통한 클래스 불균형 처리

### 14주차 - 의료비 예측
**파일:** `14주차실습.ipynb`
- 데이터셋: 국민건강보험 진료내역 2023 (1,300만+ 건)
- 대규모 의료 데이터 샘플링 및 전처리
- 환자 본인부담률 예측을 위한 회귀 분석
- LightGBM을 활용한 회귀 및 다중분류 모델 구현

## 사용 기술

- **Python 라이브러리:** pandas, numpy, scikit-learn, matplotlib, seaborn
- **머신러닝 프레임워크:** XGBoost, LightGBM
- **자연어 처리:** KoNLPy (한국어 NLP)
- **시각화:** matplotlib, seaborn

## 데이터셋

- Titanic Dataset (Kaggle)
- Breast Cancer Dataset (sklearn)
- Wine Dataset (sklearn)
- Insurance Dataset
- NSMC (네이버 영화 감성 분석 말뭉치)
- Cardiovascular Disease Dataset (Kaggle)
- 건강검진 데이터 2024 (공공데이터포털)
- 국민건강보험 진료내역 2023 (국민건강보험공단)
